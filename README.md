# Accurate Industries AI Platform

AI-powered industrial manufacturing platform built using FastAPI, MySQL, HTML, CSS, JavaScript, and Retrieval-Augmented Generation (RAG).

## Features

- AI Manufacturing Chatbot
- FAISS Vector Search
- Session Memory
- Product Management System
- Inquiry Management
- Admin Dashboard
- Responsive UI
- MySQL Database

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- FastAPI
- SQLAlchemy
- MySQL

### AI & ML
- Sentence Transformers
- FAISS
- Intent Classification
- RAG Architecture
